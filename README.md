# Onboarding AI

A continuously learning, role-aware onboarding companion that captures tribal knowledge
and improves with every new hire.

## Features
- Role-specific onboarding guidance
- RAG-based answers with source citations
- Slack + Web chat
- Feedback-driven learning loop
- Privacy-safe, team-level memory

## Stack
- Frontend: React + TypeScript
- Backend: FastAPI + Python
- Vector DB: pgvector / Pinecone
- Integrations: Slack, GitHub, Jira
